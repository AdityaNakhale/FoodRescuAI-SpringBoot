#FoodRescuAI-SpringBoot
##Food Rescue AI is a smart web-based platform that connects food donors with nearby NGOs to reduce food wastage. 
It uses location-based intelligence (latitude & longitude) to allocate surplus food efficiently. Donors can post food details, 
while NGOs receive optimized assignments based on distance and capacity. The system is built using Spring Boot, MySQL, and Thymeleaf. 
It ensures fast, fair, and real-time food distribution. The goal is to create an efficient food rescue network and support communities in need.

#Technologies Used
  -Frontend: HTML, CSS, JavaScript, Thymeleaf
  -Backend: Java, Spring Boot
  -Database: MySQL
  -Tools & APIs: Google Maps API (for location handling), REST APIs

#Key Features
🔐 Secure login system for Donors and NGOs
📍 GPS-based location tracking (Latitude & Longitude)
🤖 Intelligent food allocation algorithm
📦 Real-time food donation posting and management
📊 Dashboard for tracking donations and requests
📁 File upload support (for NGO verification)
⚡ Fast backend processing using REST APIs
